# Pkg_Basic
This tool is created for beginner termux users, he can install all basic required packages in termux
# Installation
```
pkg update
pkg upgrade
pkg install git
git clone https://github.com/MrHacker-X/Pkg_Basic.git
cd Pkg_Basic
chmod +x install.sh
```
## run
```sh install.sh```
